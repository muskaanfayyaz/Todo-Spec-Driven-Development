# Phase IV: AI Agents Usage

This document outlines the usage and constraints for AI agents within Phase IV.

## Permitted Agents
- Gemini CLI (acting as the primary orchestrator)
- Claude Code (for code generation and analysis, as instructed by Gemini CLI)

## Prohibited Agents
- `kubectl-ai`
- `kagent`

## Interaction Guidelines
- All modifications to the codebase and infrastructure configurations must be initiated and guided by the approved AI agents.
- No manual code edits are permitted outside of the specified AI agent interactions.
- AI agents must adhere to the defined goals and constraints of Phase IV.
