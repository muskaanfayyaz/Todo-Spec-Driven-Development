# Claude Code Usage in Phase IV

This document details the specific role and interaction protocols for Claude Code within Phase IV, as orchestrated by Gemini CLI.

## Role
Claude Code will be utilized for:
- Generating Dockerfiles for backend and frontend services.
- Assisting in the creation of Helm chart templates.
- Providing code snippets for Kubernetes manifests where necessary.
- Analyzing existing Phase III code to inform containerization strategies.

## Interaction Protocol
- Gemini CLI will provide specific prompts and context to Claude Code for generating or analyzing content.
- Claude Code's output will be reviewed and integrated by Gemini CLI into the `phase4` directory.
- All outputs from Claude Code must align with the overall goals and constraints of Phase IV.
